# Contributing to Pridge

Pridge is an ecosystem of independently versioned repositories. Open issues and pull requests in the repository that owns the affected component.

## Repository responsibilities

- `Pridge`: ecosystem index, website and shared project documentation
- `PrintBridge-Server`: server, API, queue and administration
- `PrintBridge-Client`: desktop/headless client and local printing
- `Pridge-Dolibarr-Endpoint`: Dolibarr Receipt Printers and TakePOS integration

Keep changes focused, document behavior changes and never commit credentials or private print payloads.
